# AMBRE SHOP

Site vitrine démo pour une marque de cosmétiques premium.

## Stack technique

- Next.js 15 (App Router)
- TailwindCSS 4
- TypeScript
- Export statique

## Lancer le projet

```bash
npm install
npm run dev
```

Le site sera accessible sur [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Modifier le numéro WhatsApp

Éditez le fichier `src/config/brand.ts` :

```ts
whatsappNumber: "24100000000",
```

## Pages

| Route        | Description                |
| ------------ | -------------------------- |
| `/`          | Page d'accueil             |
| `/besoin`    | Présentation de l'offre    |
| `/facebook`  | Simulation page Facebook   |
| `/whatsapp`  | Simulation chat WhatsApp   |
| `/catalogue` | Catalogue produits         |
